## Learnings
- AVAudioFormat initializer on macOS 26.5 requires `channelLayout: AVAudioChannelLayout` instead of `channelCount: Int`
- AVAudioChannelLayout(layoutTag:) returns optional, needs force-unwrap for mono layout
- AppDelegate needs @MainActor annotation when accessing NSPopover from togglePopover
- SessionCoordinator requires full implementation of initialize(), startRecording(), stopRecording(), transcribe(), deliver(), and cleanup()
- Xcode project pbxproj must explicitly list all source files in PBXBuildFile section and PBXSourcesBuildPhase
- swift build works reliably for SPM projects; xcodebuild may have issues with SPM dependency resolution
- MLXAudioSTTProvider requires MLX, MLXAudioSTT, and MLXAudioCore imports from mlx-audio-swift package
- Button in SwiftUI requires explicit `action:` label when using closure syntax
- NSApplication.shared.terminate() requires self parameter, not nil
- AppState.permissionRequired case must be handled in all UI switch statements
